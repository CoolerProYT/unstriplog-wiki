# Getting Started

This page provides basic information for Unstrip Log v1.1.0+.

## Installation

Unstrip Log can be downloaded on CurseForge and Modrinth.

| Platform   |                                        Link                                         |
|------------|:-----------------------------------------------------------------------------------:|
| CurseForge | [Download from CurseForge](https://www.curseforge.com/minecraft/mc-mods/unstriplog) |
| Modrinth   |            [Download from Modrinth](https://modrinth.com/mod/unstriplog)            |
